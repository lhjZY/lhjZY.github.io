这是一个使用文件夹结构的文章示例。

使用文件夹结构的好处是可以将图片放在文章同目录下，方便管理。


<!--more-->

## 如何使用

1. 在 `posts/` 目录下创建一个文件夹，如 `my-article/`
2. 在文件夹中创建 `index.md` 作为文章内容
3. 将相关图片放在同一文件夹中
4. 使用相对路径引用图片：`![](./image.png)`

## 目录结构示例

```
posts/
├── my-article/
│   ├── index.md      # 文章内容
│   ├── cover.png     # 封面图
│   └── diagram.png   # 其他图片
```
